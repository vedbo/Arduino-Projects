Cloud IoT

Building a Wi-Fi Weather Station

Presented by Ved Borade | Rutgers Organization of Cloud Computing (ROCC)

Our Mission Today

We are going to bridge the physical world with the Cloud.

  • The Edge: ESP32 Microcontroller
  • The Sensor: DHT11 Temperature & Humidity
  • The Cloud: Google Firebase Realtime Database

Goal: Stream live room temperature to a web dashboard natively over Wi-Fi!

The Hardware

What's sitting in front of you.

1. The ESP32 (The Brain)

  • Dual-Core 240MHz Processor (Very fast for a microcontroller!)
  • Built-in Wi-Fi & Bluetooth
  • 3.3V Logic level

2. The DHT11 (The Eyes)

  • Digital humidity and temperature sensor.
  • Only uses 3 pins!
  • Extremely beginner-friendly.

Wiring It Up

Grab your 3 jumper wires. We do this before anything else!


DHT11 Pin ESP32 Pin
VCC / + → 3V3 (Power)
GND / - → GND (Ground)
DATA / OUT → GPIO 15 (or D15)

The Cloud: Firebase

Google's Backend-as-a-Service.

Creating the Database

  1. Go to console.firebase.google.com
  2. Create a project: "ROCC Workshop ESP32"
  3. Navigate to Realtime Database
  4. Click Create Database and select Start in Test Mode.

Securing the Connection

For today's workshop, we are bypassing authentication so everyone can connect immediately.

Go to your Database Rules tab and ensure it says:

{
  "rules": {
    ".read": true,
    ".write": true
  }
}

The Missing Link

To connect the ESP32, you will need to copy the Database URL from the top of the Data tab.

Reminder: Strip off the 'https://' and any trailing '/' when pasting into your code!

The Software Setup

Preparing the Arduino IDE.

  1. Make sure your board is set to: DOIT ESP32 DEVKIT V1.
  2. Go to Sketch → Include Library → Manage Libraries.
  3. Install "DHT sensor library" by Adafruit.
  4. Install "Firebase ESP32 Client" by Mobizt.

The Magic Block

Open the provided code template and fill in the blanks!

#define WIFI_SSID "RUatHome"
#define WIFI_PASSWORD "********"

#define DATABASE_URL "YOUR_DATABASE_URL_HERE.firebaseio.com"

// The ESP32 pushes the data structure below to the cloud automatically!
FirebaseJson json;
json.set("temperatureF", temperatureFahrenheit);
json.set("humidity", humidity);
Firebase.setJSON(fbdo, "/rocc_sensor_node", json);

Upload & Verify!

  • Click the Upload button ➡️
  • Wait for "Connecting...", then it will begin flashing.
  • Open your **Serial Monitor** (set baud rate to `115200`).
  • Open your Firebase Console side-by-side.

Watch the data stream live!
